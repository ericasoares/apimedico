# Sobre o projeto

Api Medico é um microserviço de aplicação backend Spring Boot Java.

# Tecnologias utilizadas
## Back end
- Spring Boot 3
- Java 17
- JPA / Hibernate
- Maven
- Migration / Flyway
- Banco de dados: MySQl 


Pré-requisitos:
- Java 17
- MySQL
